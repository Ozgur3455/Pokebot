# POKEMON BOTU
Projemde pokemona sahip olarak başkalarıyla savaşmayı, canı azaldığında besleyerek canını yenilemeyi ve her pokemonun sahip oldu sınıfa göre can aldığı miktarın değişmeside bulunmaktadır. Arkadaşlarınızla pokemon sahibi olarak savaşabilir keyifli zaman geçirebilirsiniz.
